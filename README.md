# Researcher Personal Website Template

This is a Jekyll-based template for a personal website for researchers.

## Setup

1. Install Ruby and Bundler if not already installed.
2. Clone or download this repository.
3. Run `bundle install` to install dependencies.
4. Customize the content in `index.md`, `publications.md`, and `cv.md`.
5. Update `_config.yml` with your information.
6. Run `bundle exec jekyll serve` to preview the site locally.
7. Deploy to GitHub Pages or your preferred hosting.

## Structure

- `index.md`: Home page with biography, research interests, and news.
- `publications.md`: List of publications.
- `cv.md`: Curriculum vitae with education and employment history.
- `_config.yml`: Jekyll configuration.

## Customization

Edit the markdown files to replace placeholder content with your actual information.
